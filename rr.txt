Отлично, поработал с материалами. Вот что выяснил и как это соотносится с твоей схемой:

## Что такое SH847 на самом деле

SH847 (Shindengen) — это **serial (series/open-type) регулятор-выпрямитель на 6 тиристорах**. Ключевой момент — он не использует отдельный диодный мост + ключ. Он использует **6 тиристоров (SCR) как сам управляемый выпрямительный мост**.

**Принцип SH847:**
```
Фаза A ──→ SCR1 ──→ +выход
Фаза B ──→ SCR3 ──→ +выход    ( SCR = управляемый диод)
Фаза C ──→ SCR5 ──→ +выход       Упр.вход → логика по Uбатареи
              ↕                Логика задерживает угол открывания
Фаза A ──→ SCR4 ──→ GND        тиристоров = меньше энергии проходит
Фаза B ──→ SCR6 ──→ GND        статор почти не нагружен
Фаза C ──→ SCR2 ──→ GND
```

- Когда батарея не заряжена → тиристоры открываются в начале полупериода → полная выпрямка → максимальный заряд
- Когда батарея заряжена → логика **задерживает угол открывания** тиристоров → часть полупериода **блокируется** → статор в этот момент **обесточен** → нет нагрузки
- Тиристоры **естественно коммутируют** на нулевом переходе — нет проблемы с разрывом тока

**Почему статор не сгорает:** Тиристор либо проводит (ток идёт в батарею), либо блокирует (ток через статор = 0). Нет режима КЗ. Каждый тиристор рассчитан на 600-1000В, так что пиковые напряжения при обрыве цепи для них не страшны.

## Разбор твоей схемы

Твоя цепочка:
```
Статор → Мост 50А → Конденсатор 250В → TVS → MOSFET 4110 → Конденсаторы 20В → Батарея
```

### Что здесь правильно ✅

1. **Идея последовательного (series) регулятора** — абсолютно верная. SH847 доказал, что это лучший подход для ПМ-генераторов. Статор не греется, КПД высокий, мощность двигателя не отбирается впустую.

2. **MOSFET вместо тиристора** — в принципе хорошая идея. IRF4110 имеет Rds(on) ≈ 3.7 мОм. При 20А потеря = 0.004 × 20 = **0.08В, мощность ~1.6Вт**. Это ничто по сравнению со шунтирующим регулятором. Ты прав — КПД будет бешеный.

3. **TVS-диоды для защиты** — разумно, но есть нюансы (ниже).

4. **Выходные конденсаторы 20В** — правильный подход для сглаживания пульсаций после ключа.

### Критические проблемы ❌

**1. Напряжение на конденсаторе и MOSFET — ГЛАВНАЯ ПРОБЛЕМА**

Когда MOSFET закрыт, статор через диодный мост заряжает конденсатор до **пикового напряжения**. Для трёхфазного генератора:
```
Uпик_DC ≈ Uфазное_АЦ × √2 ≈ 40-200В АЦ (зависит от оборотов) × 1.41
```

На холостых/высоких оборотах без нагрузки статор легко выдаёт 100-200В АЦ на фазу, значит на конденсаторе будет **140-280В DC**.

**IRF4110: Vds(max) = 100В** — он **сгорит мгновенно** при первом же открытии, когда конденсатор заряжен до 200В.

**2. TVS-диоды не спасут (или станут шунтирующим регулятором)**

Если ты поставишь TVS на, скажем, 80В чтобы защитить MOSFET:
- Весь избыточный ток статора пойдёт через TVS → они будут **рассеивать ту же мощность, что и шунтирующий регулятор**
- TVS на 250В не защитят MOSFET (100В)

Получается замкнутый круг: либо MOSFET сгорает, либо TVS превращает схему обратно в шунтирующий регулятор.

**3. Нет индуктивности между ключом и конденсатором**

Если MOSFET работает в ШИМ-режиме (а не как простое реле вкл/выкл), без дросселя будут колоссальные импульсные токи при каждом включении — конденсатор 250В заряжен, батарея 14В, разница 200+В, даже ESL проводов даст огромный di/dt.

**4. Проблема разрыва тока**

При выключении MOSFET ток через индуктивность статора не может мгновенно исчезнуть. Без снаббера/варистора на статорной стороне будет дуговой пробой или выброс напряжения, который добьёт и мост, и конденсатор.

## Как сделать правильно — два реальных пути

### Путь А: Повторить логику SH847 (управляемый выпрямитель на тиристорах)

```
Статор (3 фазы)
  ↓
6 тиристоров (управляемый мост) ← Логика управления (по Uбатареи)
  ↓                            задержка угла открывания
  + Выход на батарею (через конденсаторы сглаживания)
```

**Детали:**
- 6 шт. тиристоров типа **BT151** (16А, 800В) или **BT152** (20А, 800В) — копейки
- Управляющая логика: компаратор (LM393) + схема синхронизации с фазами (через маломощные трансформаторы или оптроны) + генератор задержки
- Выходные конденсаторы 20-50В для сглаживания
- **Минимум деталей**, высокая надёжность
- Тиристоры естественно закрываются при переходе тока через ноль — нет проблемы коммутации
- Статор либо проводит в батарею, либо полностью обесточен

**Минус:** Нужна синхронизация с фазами для управления углом открытия тиристоров. Это усложняет логику, но не усложняет силовую часть.

### Путь Б: Твоя идея, но с правильным MOSFET и дросселем (понижающий преобразователь)

```
Статор → Мост 50А → Конденсатор (среднее звено) → Дроссель → Диод → Батарея
                                  ↑                         ↑
                              MOSFET (вниз на GND)    Конденсатор 20В
                                  ↑
                           ШИМ-контроллер (по Uбатареи)
```

Это классический **buck converter** (понижающий преобразователь). MOSFET стоит **низом на массу**, а не последовательно.

**Детали:**
- Мост 50А (6 диодов) — OK
- **MOSFET: нужен на 300-500В**, например **IRFP460** (500В, 20А, 0.27Ω) или **FQP9N50** (500В, 9А) — можно параллелить
- **Дроссель:** обязательно! ~100-200 мкГн, на ток 40-50А. Можно намотать на ферритовом кольце
- **Диод Шоттки** обратный (или быстрый диод) после дросселя — чтобы ток статора не обрывался
- ШИМ-контроллер (можно на TL494, UC3843, или даже простом 555 + компаратор)
- Входной конденсатор: 250-400В номинал, ёмкость 100-470 мкФ
- TVS/варистор на входе для защиты от экстремальных выбросов

**Плюсы:** КПД 90-95%, статор почти не нагружен, чистый DC на выходе
**Минусы:** Больше деталей (дроссель, обратный диод, ШИМ-контроллер), высоковольтный конденсатор дорогой

## Сравнение твоего варианта с SH847

| Параметр | SH847 (6 тиристоров) | Твой вариант (мост + MOSFET) | Твой вариант (исправленный) |
|---|---|---|---|
| Нагрузка на статор | Минимальная | Минимальная (в теории) | Минимальная |
| КПД | Высокий (~90%) | ❌ MOSFET сгорит | 90-95% |
| Количество деталей | ~15-20 | ~10 (но нерабочий) | ~20-25 |
| Сложность сборки | Средняя (нужна синхронизация) | Низкая (но не работает) | Средняя |
| Надёжность | Очень высокая | ❌ | Высокая |
| Проблема холостого хода | Нет (тиристоры блокируют) | ❌ 280В на 100В MOSFET | Решено (MOSFET на 500В) |
| Тепловыделение | Низкое | — | Низкое |

## Моя рекомендация

Твоя идея с MOSFET-ом — **правильная по сути**, но нужна доработка:

1. **Либо** замени MOSFET 4110 на высоковольтный (IRFP460, 500В) и добавь дроссель с обратным диодом — получится полноценный buck-конвертер. Это даст наилучший КПД, но деталей будет больше.

2. **Либо** иди по пути SH847 — 6 тиристоров вместо диодного моста + MOSFET-а. Деталей меньше (нет высокого конденсатора, нет дросселя), надёжность выше, но нужна логика синхронизации с фазами.

Для ручной сборки с минимумом деталей я бы склонялся к **варианту А (тиристорный управляемый мост)** — это ближе всего к SH847, минимум силовых деталей, и статор вообще не нагружается когда не нужно.
